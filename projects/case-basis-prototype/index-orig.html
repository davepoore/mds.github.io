ACTIONS


Action 1 - External

Links to something outside of the app. Eg. Google Maps

Action 2 - Modal Launch

Launches a modal—usually a form–where a user must decide to cancel the transaction or save new information

Action 3 - Status Update

Creates a status update "stamp" on the messages timeline

Action 4 - Drilldown

Links to a detail view of specific information. Return to main section via "Back" button

Action 4.1 - Drilldown + Signature

Inclusion of an embedded client signature. Used to represent signed documents
